Elements:
> Natives (no func.)
> Inputs (create func.)
> Variables (receive func.)
> Grouped Elements (control func.)

ALL ELEMENTS:
> Have HTML
> Have width & height

Element Styling:
> Set a default element style (ie. color = white by default)
> So that if you remove the style it will go to the default

Element Sets:
> Group elements for different uses (barebones elements vs. fully functional elements / pre-styled)
